**API for the useUtils function**

`useUtils` is a function that returns some helpful utilities for your query.

## Usage

```tsx
const utils = testQuery2.useUtils()
```

### .invalidate

This function should be used to re-validate your query data (usually done after mutations calls).

```ts
const helloRes1 = testQuery2(() => ({
  hello: 'jdev',
}))
const helloRes2 = testQuery2(() => ({
  hello: 'jdev22',
}))

const helloMutation = testMutation(() => ({
  async onSuccess() {
    console.log('invalidate all testQuery2')
    await utils.invalidate()

    // invalidate only  jdev
    await utils.invalidate({ hello: 'jdev' })

    // invalidate only jdev22
    await utils.invalidate({ hello: 'jdev22' })
  },
}))
```

You can invalidate all query calls, or just a query with a certain input.
