**Adding MediaKit/forms to SolidStart**

## Install

```sh
pnpm add @solid-mediakit/forms
```

Check out the [`createForm`](/forms/createform) function.

## Supports

- [x] String
- [x] Integer
- [x] Float
- [x] Boolean (Checkbox)
