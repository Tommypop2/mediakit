**Adding MediaKit/forms to SolidStart**

## Install

```sh
pnpm add @solid-mediakit/forms
```

client actions = Check out the [`createForm`](../createform) function.
server actions = Check out the [`createForm$`](../serverform) function.

## Supports

- [x] String
- [x] Integer
- [x] Float
- [x] Boolean (Checkbox)
